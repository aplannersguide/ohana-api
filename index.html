Road Map

The Ohani API is a project from the San Mateo County team of Code for America fellows that is aiming to create open access to community social services, with an initial emphasis on food security.

The planned development roadmap of the Ohana API involves the creation of the read/write API as well as demonstration apps showing what's possible with the API.

  1. May 2013: Demonstration app: Develop demonstration front-end application that can query the database.
    View Demonstration Application Prototype.
  2. June 2013: Readable API: Develop Ohana database that returns read queries over HTTP.
    View Project on GitHub.
  3. June 2013: Admin API interface: Develop admin application that has read/write access to update database data.
  4. Later 2013: Community-built innovation/apps: Additional applications, such as an SMS (text messaging) search interface for database could be developed to use the data provided by the Ohana API.
